# sjp.py

Word definition extractor from website SJP.pl which is popular polish dictionary. Written in Python 3. Uses BeautifulSoup library.

## Usage

	sjp.py <word>
	sjp.py (-h | --help | /?)
	sjp.py (-v | --version)

## Dependencies

* Python 3
* [BeautifulSoup](http://www.crummy.com/software/BeautifulSoup/)
